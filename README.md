
# BusRoute-ABM

## Overview
A bus route agent-based model (ABM) for simulating and analyzing public transportation systems.

## Features
- Agent-based modeling of bus routes and passengers
- Real-time route simulation
- Performance metrics and analytics

## Getting Started

### Prerequisites
- Python 3.8+
- Required dependencies (see `requirements.txt`)

### Installation
```bash
git clone https://github.com/kavyasukumar/ABM-helloworld.git
cd ABM-helloworld
pip install -r requirements.txt
```

### Usage
```bash
python run.py
```
